# excel-production-dashboard
## Overview
An interactive Excel dashboard built to gain hands on experience on xlookup,vlookup,slices and dynamic filtering

## Key Features
- Interactive slicers for dynamic filtering
- Pivot Tables and Pivot Charts

## Data Transformation & Cleaning
- Used VLOOKUP and XLOOKUP
- Checked for missing/null values
- Replaced null values with "Unknown"
- Created calculated fields for analysis

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- VLOOKUP
- XLOOKUP
- Data Cleaning
- Dashboard Design
